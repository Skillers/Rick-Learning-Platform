In Python gebruik je loops om code herhaaldelijk uit te voeren. Er zijn twee soorten: de for-loop en de while-loop.

For-loop

De for-loop itereert over een reeks waarden, zoals een lijst of een range().

# Print getallen 1 tot 5 for i in range(1, 6): print(f"Getal: {i}") # Output: # Getal: 1 # Getal: 2 # Getal: 3 ...

Over een lijst itereren

namen = ["Anna", "Boris", "Clara"] for naam in namen: print(f"Hallo, {naam}!")

While-loop

De while-loop blijft draaien zolang een conditie waar is. Let goed op dat je geen oneindige loop maakt!

teller = 0 while teller < 5: print(f"Teller: {teller}") teller += 1 # BELANGRIJK: vergeet dit niet!
💡 Handige loop-commando's

break — stop de loop onmiddellijk
continue — sla de rest van deze iteratie over
else — voer dit uit als de loop normaal eindigt

Geneste loops

Je kunt loops ook in elkaar nesten:

for rij in range(3): for kolom in range(3): print(f"({rij},{kolom})", end=" ") print() # nieuwe regel
