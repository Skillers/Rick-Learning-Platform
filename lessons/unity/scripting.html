In Unity schrijf je gedrag voor GameObjects met C# scripts. Elk script erft van MonoBehaviour — de basis-klasse voor alle Unity-scripts.

Een script aanmaken

Ga naar Assets → Create → C# Script en geef het een naam. Dubbelklik om het in Visual Studio of Rider te openen.

using UnityEngine; public class MijnScript : MonoBehaviour { // Wordt eenmalig uitgevoerd bij het starten void Start() { Debug.Log("Het spel is gestart!"); } // Wordt elke frame aangeroepen (~60x per seconde) void Update() { // Hier schrijf je gedrag dat elke frame gebeurt } }

GameObject bewegen

Met transform.position kun je een object verplaatsen. Gebruik altijd Time.deltaTime zodat beweging framerate-onafhankelijk is.

public class PlayerMovement : MonoBehaviour { public float speed = 5f; void Update() { float h = Input.GetAxis("Horizontal"); // A/D of pijltjes float v = Input.GetAxis("Vertical"); // W/S of pijltjes Vector3 richting = new Vector3(h, 0, v); transform.position += richting * speed * Time.deltaTime; } }
⚠️ Belangrijke regel

Vergeet Time.deltaTime nooit bij beweging! Zonder dit beweegt je object 60× sneller op 60fps dan op 1fps.

Public variabelen & de Inspector

Public variabelen zijn zichtbaar en aanpasbaar in de Unity Inspector — ideaal voor het testen van waarden zonder opnieuw te compileren.

public float jumpForce = 10f; // Zichtbaar in Inspector private float health = 100f; // Niet zichtbaar in Inspector

🎯 Opdracht 1 — Roterend object

Maak een nieuw C# script Rotator. Voeg een public float rotationSpeed toe met waarde 90. Gebruik in Update(): transform.Rotate(0, rotationSpeed * Time.deltaTime, 0); Koppel het script aan een Cube in je scene.

🎯 Opdracht 2 — Beweging met input

Kopieer het PlayerMovement script hierboven. Pas het aan zodat het object ook omhoog/omlaag kan bewegen met de Q en E toetsen. Tip: gebruik Input.GetKey(KeyCode.Q)

✅ Goed gedaan!

Sla je scripts op en test ze in de Unity Editor. Kijk goed naar de Console voor eventuele foutmeldingen. Als alles werkt, ga dan door naar Physics & Colliders.
